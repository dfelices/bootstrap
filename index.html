What Is Bootstrap?

Created at Twitter in mid 2010s, Bootstrap is a front-end framework program that excels in responsiveness and interactivity.

Learn more

What Is Bootstrap?

Bootstrap uses HTML and CSS predesigned templates for typography and color styles for UI components including buttons, navigation, image carousels, and more.

Learn more

What Does Bootstrap Do?

Imagine the framing of a house, it makes building quicker and more efficient. Bootstrap is a framework that helps streamline building responsive page

Learn More

Why use it?

Bootstrap primary use is to quickly develop user-friendly websites that can be visually appealing but also accessible for all users.

  • Readily available resources
  • Prototyping
  • Responsiveness
  • Easy to use

Bootstrap has use cases where consistency is a key factor on developing uniform designs across multiple frameworks. Other uses of Bootstrap include.

  • Open source
  • Browser compatibility
  • JavaScript Integration

Other frameworks

  • Foundation
  • Tailwind CSS
  • UIkit
  • Semantic UI
  • Materialize
  • And more...but Bootstrap is one of the most popular!

Where would you use it?

Basically, you can use it for your entire site! It's a framework, “framing for your house”.

  • Headers
  • Nav bars
  • Hero sections
  • Sidebars
  • Footers
  • Dropdowns
  • Typography sizing
  • Even a pre-built colour palette!

Pros

  • Building with responsive design in mind right from the beginning.
  • Quicker, more efficient building.
  • Consistent web pages.
  • Great for building with teams.
  • Widespread adoption (many resources, community support.)
  • User-friendly and comprehensive documentation.
  • Many built-in components available.
  • But also highly, customizable!

Cons

  • Uniformity leads to cookie cutter designs.
  • File Size affects the load time and overall user experience on the website.
  • Learning curve takes a while to learn all the classes.
  • Highly customizable but a hassle to change.
  • Can get highly dependent.
  • Versioning changes are quite different.
